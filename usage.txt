Usage: spritesheet-assembler [arguments]

Arguments:
  -i, --input-folder        path to the input folder with sprites to assemble
  -o, --output-image        path to the output folder
  -d, --output-descriptor   path to the output descriptor file
  -t, --descriptor-type     type of descriptors to generate. Currently supported
                            values are:
                                1. devtools - generate devtools descriptors
  -p, --padding             padding between sprites

Examples:
    spritesheet-assembler -i icons/ -o spritesheet.svg -d spritesheet.js -t devtools
